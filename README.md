# 酒店管理系统 java21

## 项目概述

酒店管理系统是基于B/S架构和MVC设计模式的Web应用，采用前后端分离的技术方案，主要使用Java和Vue进行开发。本系统致力于提供便捷、高效的管理功能，以提升酒店的服务质量和运营效率。

## 技术栈

### 系统环境
- Windows/Mac

### 开发环境
- IDEA
- JDK 1.8
- Maven
- Mysql

### 技术组件
- Java
- MySQL
- SpringBoot
- Mybatis
- Vue

## 功能模块

### 用户模块
- 用户登录
- 用户注册
- 预约住房
- 留言
- 查看公告
- 查看预约
- 入住查询

### 管理员模块
- 管理员管理
- 用户管理
- 房型管理
- 房间管理
- 预约管理
- 入住管理
- 留言管理

## 系统角色
- 用户
- 管理员

## 运行环境
- 服务器：支持Java和MySQL运行的服务器
- 客户端：现代Web浏览器

## 部署步骤
1. 准备服务器环境（Java和MySQL）
2. 部署后端服务（部署SpringBoot应用）
3. 部署前端服务（Vue项目）
4. 启动服务
